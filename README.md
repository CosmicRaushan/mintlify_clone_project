# mintlify_clone_project

Mintliy is an AI-powered documentation platform where developers create, maintain, and publish modern software documentation.

This project is a clone of the landing web page of Mintlify, a fully functional web page of Mintlify. This clone project does not have responsiveness and is not fully functional.

This project is built solely for learning purposes using basic HTML and CSS.

When you clone this project, you might see a rearrangement of HTML elements on your device because it lacks CSS responsiveness.
